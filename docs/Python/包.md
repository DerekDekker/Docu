# 包


文件.py 为 模块

多个模块的目录并存在 __init__.py 文件为 包

```
包/
    模块.py
    模块.py
    __init__.py
    包/
        模块.py
        模块.py
        __init__.py
```

---
## init

`__init__.py` 文件下导入完 模块 之后 在引入这个包时可以直接引入具体的东西

```python title="__init__.py"
for .模块 import 名称
for .模块 import 名称
for .模块 import 名称

```


---
## all

指定哪些内容外界可以访问, 只有被 `from 包 import *` 全部引用时才有效

```python
__all__ = ['变量名', '类名', '函数名']
```
